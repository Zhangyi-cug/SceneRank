受试者对随机图像对在多个维度作出偏好判断，数据实时入库，支持 CSV 导出。

功能特性

为街道感知研究设计的完整工具链

🖼️
成对比较范式
随机抽取图像对，受试者在安全、美丽、活力等多维度逐一作出偏好判断。
📝
背景调查
可配置的背景问卷，支持下拉、单选、文本题型，收集受试者基本信息。
⚙️
管理后台
可视化配置问卷内容、维度、图片范围，实时查看数据，一键导出 CSV。
📊
数据管理
所有对比记录存入数据库，支持单条删除、清空、按时间筛选与 CSV 导出。
📱
响应式设计
适配桌面与移动端，受试者可在手机上完成问卷，无需安装任何应用。
🚀
易于部署
前后端分离，提供 Nginx + Gunicorn + systemd 完整部署配置，开箱即用。

技术栈

前后端分离架构，轻量易维护

Vue 3 Vue Router Vite Django 4 Django REST Framework SQLite / PostgreSQL Nginx Gunicorn Python 3
